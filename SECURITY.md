# Security Policy

## Supported Versions

Security fixes are applied to the active `main` branch.

| Version | Supported |
| --- | --- |
| `main` | Yes |
| older tags/branches | No |

## Reporting A Vulnerability

1. Do not open a public GitHub issue.
2. Send a private report to the maintainers.
3. Include:
   - Impact summary
   - Reproduction details
   - Potential mitigation (if known)

We will acknowledge reports as soon as possible and work on validation/remediation.
