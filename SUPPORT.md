# Support

## Where To Ask For Help

1. Usage and troubleshooting:
   - Open a GitHub Discussion (recommended), or
   - Open a GitHub Issue with logs/steps.
2. Bugs:
   - Use the bug issue template and include reproduction steps.
3. Security concerns:
   - Follow `SECURITY.md` (private reporting only).

## What To Include

- Operating system and version
- .NET SDK version (`dotnet --version`)
- Command or UI flow used
- Relevant logs/errors
- Expected vs actual behavior
